# ゲームのコアシステムと機能一覧（Phase 6地点）

## 🌟 コアシステム（盤面と操作）
1. **5x5 マージグリッド (`GridManager`)**
   - 同じレベルのアイテムを重ねる（マージする）と、1つ上のレベルに進化します。
   - **自由移動と入れ替え**: マージだけでなく、空いているスロットへの移動や、違うレベルのアイテム同士の場所の入れ替え（Swap）が自由にできます。
   - **移動ペナルティ**: アイテムを移動・入れ替えた直後は、0.5秒間スロットが暗くなり「クールダウン（攻撃不可状態）」に入ります。

## 🧟 防衛と自動攻撃システム
2. **ウェーブ進行 (`EnemyManager` & `LevelManager`)**
   - 画面上部からゾンビが防衛ライン（画面下部）に向かって進んできます。
   - ステージ制になっており、指定された数のゾンビを全滅させると「STAGE CLEAR」となり、次のステージに進みます（敵の数や移動速度が上がります）。
3. **自動攻撃 (`CombatSystem`)**
   - 盤面上の **Lv.3以上のアイテム** が武器として機能し、自動的に一番近いゾンビに向かって弾（`Bullet`）を撃ちます。武器のレベルが高いほど、弾のダメージも高くなります。
   - **列（Column）優先攻撃**: 武器は、まず「自分と同じ縦列にいるゾンビ」を最優先で狙います。

## 🔥 戦略と爽快感アップ機能
4. **マージ・バースト（範囲攻撃）**
   - アイテムをマージした瞬間、そのスロットから黄色い衝撃波（エリアダメージ）が発生し、周囲のゾンビ全員にダメージを与えます。
5. **コンボシステム (`ComboManager`)**
   - 1秒以内に連続してマージを成功させると「コンボ（x2, x3...）」が繋がります。
   - コンボが繋がると、画面が揺れる（カメラシェイク）演出と、マージ元の位置から黄背景＋黒文字でポップに跳ねる「x2 COMBO!」等のテキストが出現します。

## 📦 ステージの仕掛けとオーダー機能
6. **ORDERボタン（アイテムスプーナー）**
   - 「ORDER (Items)」ボタンを押すと、盤面の空きスロットにランダムで Lv.1（高確率）または Lv.2（低確率）のアイテムが補給されます。
7. **ステージ固有のレイアウトと障害物**
   - ステージ開始時、あらかじめ定義されたアイテムや「障害物（Xマークの黒いブロック）」が配置されます。障害物は動かせず、アイテムを置くこともできません。

## 💀 ゲームオーバーとリトライ
8. **防衛失敗ラインとマネタイズUI**
   - ゾンビに防衛ラインを突破されると進行が一時停止し、ゲームオーバーダイアログが開きます。
   - **AD（動画広告）ボタン**: 敵を全滅させて、無料でその場から再開（モック）
   - **COIN（コイン）ボタン**: 50コイン消費して、その場から再開（モック）
   - **GIVE UPボタン**: 諦めてシーンを読み込み直し、最初からやり直す
